Concept

Designed for Siemens Bangalore experience center, this tactile simulation table delivers an interactive, multi-screen experience showcasing their railway infrastructure products and solutions to prospective clients. The system features a custom simulation game utilizing 3D anamorphic content and rich infographics to visually map out how automated communication infrastructure functions to ensure completely safe railway journeys. The physical tactile table works in tandem with a front facing locomotive pilot POV screen that provides real-time contextual information about specific Siemens services. To make the B2B sales showcase fully comprehensive, the installation integrates a conversational AI agent. This allows clients to interact naturally and ask real-time questions about Siemens entire ecosystem of infrastructure services and solutions, translating complex corporate data into an engaging, user driven narrative.

Serving as the lead designer and technical artist for this Unreal Engine-powered installation, Designed the end-to-end spatial design, visual content creation, and interactive logic. I modeled and animated all high-fidelity, anamorphic 3D assets—such as trains and signaling systems engineering them to pop off the screen with striking depth. To dynamically showcase Siemens capabilities, I developed interactive troubleshooting scenarios where clients could trigger simulated track obstructions or signal failures to see how automated communication resolves threats in real time. Additionally, I crafted the complete user interface (UI) architecture for both the tactile table and the pilot POV screen, streamlining dense telemetry data into a clean, intuitive layout. Finally, utilizing Convai within Unreal Engine, I engineered the conversational AI agent, tailoring its knowledge base to provide instant, precise responses to client inquiries about Siemens infrastructure services.
