Concept

Pioneering a new era in sports media, this innovative broadcasting system leverages custom AI workflows to generate dynamic, "bullet-time" frozen moments during live IPL cricket matches. The technology captures high effort player actions such as acrobatic boundary saves and spectacular catches by dynamically revolving around the frozen moment in time. By seamlessly stitching AI generated multi angle visual data with live camera recordings, the workflow reconstructs a smooth, 360degree camera sweep of a single static moment. Utilized for live broadcasting, this advanced workflow delivers a highly engaging, matrix style perspective that allows audiences to experience the game's peak athletic moments at a whole new level.

As the lead technical architect, I built the entire generative AI pipeline from the ground up, deployed the system across all dedicated production machines for the IPL broadcast, and personally trained the technical operators. To handle the high-pressure demands of a live sports broadcast, I designed the user interface to be incredibly intuitive and streamlined. Operators simply upload the footage from Camera A and Camera B into the custom ComfyUI interface and select the exact frame that needs to be frozen in time. From that point, the backend pipeline automatically coordinates with Gemini and Kling models to process the scene. The system triggers Kling to simultaneously generate three distinct multi-angle visual outputs, allowing the operator to select the best angle instantly. Once chosen, the custom ComfyUI workflow automatically stitches and polishes the generated video frames with the live recordings, rendering a broadcast-ready file that is immediately sent to the live feed.
